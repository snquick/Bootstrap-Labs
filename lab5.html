Navigation Bars

  • A navigation bar is a navigation header that is placed at the top of the page
  • With Bootstrap, a navigation bar can extend or collapse, depending on the screen size.
  • A standard navigation bar is created with the tagÂ nav class="navbar navbar-expand{-sm|-md|-lg|-xl}"
  • To set the navbar color, choose from .navbar-light for use with light background colors, or .navbar-dark for dark background colors. Then, customize with bg-* .utilities.
  • Navigation bar items are enclosed in a list or div with the .navbar-nav class
  • Navigation bar items are specified with .nav-item and usually also .nav-link
  • Navigation links should not refer to pages at external websites
  • All pages in your website should have the same (or similar) navigation
